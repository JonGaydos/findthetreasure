Key Rules

  • 54 maximum guesses — then treasure is revealed automatically
  • Each guess: server computes distance using Haversine formula, returns distance only — coordinates never reach the browser
  • Hint unlocks after N guesses (User 1 sets N) — shown as a notification
  • Win condition: guess lands within User 1's tolerance radius → treasure pin revealed + celebration
  • Give up: confirmation prompt → treasure revealed on map
  • Game state (guesses) stored in browser localStorage — survives page refreshes
